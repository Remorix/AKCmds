# BomCmds

Missing AppKit commands for iOS: tops, tiffutil, tiff2icns, pbcopy, pbpaste, textutil, open

### Current Status

Aligned with AKCmds-345.60.106 (Darwin 22) for macOS implementations, iOS supportings are rewrites since there's no AppKit on iOS.
Minimal target has not been tested yet, but should at least supporting Darwin 19 (macOS 10.14 / iOS 13), backports welcomed.
